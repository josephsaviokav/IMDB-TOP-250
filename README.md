This is a responsive and interactive web application that displays the Top 250 movies from IMDb. It fetches real-time data using the IMDb API and presents it in a visually appealing, searchable format.

🚀 Features
✅ Displays the Top 250 IMDb Movies with posters, ratings, and titles.

🔍 Search bar to quickly find any movie in the list.

🧭 Click on a movie to view more details (e.g., summary, release year, rank).

📱 Fully responsive design – works on desktop and mobile.

⚡ Fast and lightweight frontend built with React.

🛠️ Technologies Used
React – for building the user interface

IMDb API – to fetch movie data

CSS – for styling
